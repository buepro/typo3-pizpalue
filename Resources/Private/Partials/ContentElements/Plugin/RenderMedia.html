==================================================================================================================== This partial might be handy in external plugins to render their media as standard media. Parameter notes: - file: Defined by bootstrap_package (see file view helpers). For images the types `\TYPO3\CMS\Core\Resource\File` or `\TYPO3\CMS\Core\Resource\FileReference` are accepted. - data: Contains content element related information on how to render the media. For example if it should be enlargeable. - settings: Contains more general information on how to render the media. For example if lazy loading should be used. - variants: Holds information about the content available with for each screen category. Might be loaded from register (`{v:variable.register.get(name: 'variants') -> f:variable(name: 'variants')}`) The `_structureMultiplier` is updated by the structure elements (e.g. provided by the extension container) and holds information about the content with reduction the parent structure elements introduced. ==================================================================================================================== {v:variable.register.get(name: 'structureMultiplier') -> f:variable(name: '_structureMultiplier')} {bk2k:data.imageVariants(as: 'variants', variants: variants, multiplier: _structureMultiplier) -> f:if(condition: _structureMultiplier)}
