==================================================================================================================== This partial might be handy in external plugins to render their media as standard media. Parameter notes: - file: Defined by bootstrap_package (see file view helpers). For images the types `\TYPO3\CMS\Core\Resource\File` or `\TYPO3\CMS\Core\Resource\FileReference` are accepted. - data: Contains content element related information on how to render the media. For example if it should be enlargeable. - settings: Contains more general information on how to render the media. For example if lazy loading should be used. - variants: Holds information about the content width available for each screen category. ==================================================================================================================== {pp:structure.variants(as: 'variants') -> f:if(condition: '!{variants}')}
